# 论文格式辅助工具

## 项目简介
这是一个面向大学生的毕业论文格式辅助工具，帮大家快速解决格式规范、版本更新带来的修改难题。

## ✨ 核心功能
- 一键适配学校论文格式模板
- 多版本格式文件对比与差异提示
- 格式问题智能检查与修正建议

## 🛠️ 本地运行方式
直接在浏览器打开 `thesis-format.html` 文件，即可使用全部功能，无需额外依赖。

## 🤖 TRAE SOLO 开发过程
本项目使用 TRAE SOLO 开发完成：
1.  需求阶段：用 SOLO 生成了基础的 HTML 页面结构
2.  功能实现：让 SOLO 帮我写了格式解析和对比的核心代码
3.  调试优化：通过对话不断优化了界面和交互体验，解决了格式文件兼容性问题

## 后续规划
后续计划上线更多格式模板和自动导出功能，帮助更多同学高效完成论文修改。
